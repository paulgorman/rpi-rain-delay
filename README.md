# Home irrigation rain delay & manual pause tool

* Raspberry Pi B+
* SainSmart 2-Channel Relay Module on RPi pin 7
* uses "wiringPi" from https://projects.drogon.net/raspberry-pi/wiringpi/

## Usage

* `water status`
* `water on`
* `water off`
